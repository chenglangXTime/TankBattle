# TankBattle 沙漠坦克大战

面向 7-11 岁小朋友的浏览器坦克对战小游戏。红方和蓝方两辆坦克在沙漠中对轰，玩家可以调整炮管角度、发射炮弹、使用有限弹药的重机枪，并在驾驶舱视角中体验沙尘暴干扰。

## 在线试玩

[https://chenglangxtime.github.io/TankBattle/](https://chenglangxtime.github.io/TankBattle/)

## 游戏特色

- 红方、蓝方两辆坦克实时对战，坦克颜色清晰区分。
- 每辆坦克有 15 格生命值，被命中 15 次后失败。
- 炮管角度可调，支持抛物线炮弹射击。
- 每辆坦克配备重机枪，弹药限制为 30 发。
- 炮击、机枪、命中、爆炸和胜利均有音效。
- 支持主战场视角和坦克驾驶舱视角。
- 沙漠战场中每隔 10 秒出现一次沙尘暴，驾驶舱视野会受到明显干扰。
- 无需安装依赖，直接用浏览器即可运行。

## 操作方式

### 红方坦克

| 操作 | 按键 |
| --- | --- |
| 左右移动 | `A` / `D` |
| 调整炮管 | `W` / `S` |
| 发射炮弹 | `F` |
| 发射机枪 | `G` |

### 蓝方坦克

| 操作 | 按键 |
| --- | --- |
| 左右移动 | `←` / `→` |
| 调整炮管 | `↑` / `↓` |
| 发射炮弹 | `L` |
| 发射机枪 | `K` |

页面下方也提供了屏幕按钮，适合触屏或不熟悉键盘的小朋友使用。

## 本地运行

方式一：直接双击 `index.html` 打开。

方式二：启动本地静态服务。

```bash
python3 -m http.server 8765
```

然后在浏览器打开：

```text
http://127.0.0.1:8765/
```


## 技术说明

- 使用原生 HTML、CSS 和 JavaScript 开发。
- 使用 Canvas 绘制沙漠战场、坦克、炮弹、驾驶舱和沙尘暴。
- 使用 Web Audio API 生成炮击、机枪、命中和环境风声。
- 无第三方依赖，适合部署到 GitHub Pages。

## 部署

本项目已配置为 GitHub Pages 静态站点，发布源为 `main` 分支根目录。
